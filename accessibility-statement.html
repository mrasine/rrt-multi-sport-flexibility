Accessibility Statement

Last Updated: March 23, 2026

Our Commitment

Rhythmic Ribbon of Texas is committed to ensuring digital accessibility for people with disabilities. We strive to make our website accessible to all users, including those who use assistive technologies.

Accessibility Features

Our website includes the following accessibility features:

  • Semantic HTML structure with proper heading hierarchy
  • Alt text for meaningful images
  • Keyboard navigation support for all interactive elements
  • Sufficient color contrast ratios for text readability
  • Font sizes that can be resized using browser controls
  • Skip navigation links for screen reader users
  • Labels and instructions for form inputs
  • ARIA landmarks and roles where appropriate
  • Focus indicators for interactive elements
  • No content that flashes more than 3 times per second

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. These guidelines explain how to make web content more accessible for people with disabilities and more user-friendly for everyone.

While we work toward full conformance, you may notice some areas that are not yet fully accessible. We are continuously improving our website's accessibility.

Known Limitations

We are aware of the following areas where accessibility could be improved:

  • Some older images may not have complete alt text
  • Third-party content (such as Formspree forms) may not be fully under our control
  • Video content does not yet include captions
  • Some interactive elements may benefit from additional keyboard support

Assistive Technology Compatibility

Our website is designed to be compatible with:

  • Screen readers (NVDA, JAWS, VoiceOver, TalkBack)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation

If you are not satisfied with our response to a complaint regarding the accessibility of our website, you may submit feedback or file a complaint through one of the following channels:

  • Contact us directly using the information above
  • If you encounter accessibility issues, please describe them in as much detail as possible

Technical Information

Accessibility of this website relies on the following technologies:

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets)
  • JavaScript (for interactive elements)
  • ARIA (Accessible Rich Internet Applications)

These technologies are used in combination with assistive technology to conform to accessibility standards.

Third-Party Content

Our website includes links to third-party content and services. While we encourage third-party providers to provide accessible content, we cannot guarantee the accessibility of external sites.

Ongoing Efforts

We are continuously working to improve the accessibility of our website. This includes:

  • Regular audits of our website using automated and manual testing tools
  • Incorporating accessibility into our design and development processes
  • Training our team on web accessibility best practices
  • Reviewing and updating this accessibility statement periodically
